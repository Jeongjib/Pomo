# 🍅 Pomo 설치 안내

뽀모도로 타이머 + 자리 비움 자동 감지 macOS 앱.

---

## 설치

### 1. 압축 풀기
받은 `Pomo.zip`을 더블클릭하면 같은 폴더에 `Pomo.app`이 나옵니다.

### 2. Applications 폴더로 이동 (권장)
`Pomo.app`을 **응용 프로그램(Applications) 폴더**로 드래그하세요.

### 3. 첫 실행 — ⚠️ 그냥 더블클릭하면 안 열려요

이 앱은 Apple 공식 인증을 거치지 않아서 macOS가 처음에는 차단해요.
한 번만 다음 절차로 우회하면 그 다음부터는 그냥 더블클릭으로 열립니다.

**일반적인 경우 (macOS 14 이하)**
1. `Pomo.app` **우클릭** → **"열기"** 선택
2. 다이얼로그가 뜨면 **"열기"** 한 번 더 클릭

**macOS 15(Sequoia) 이상**
위 방법으로 안 열릴 수 있어요. 그러면:
1. 한 번 더블클릭 시도 (차단 메시지 떠도 OK)
2. **시스템 설정 → 개인정보 보호 및 보안** 열기
3. 스크롤 내려서 **"Pomo이(가) 차단되었습니다"** 메시지 찾기
4. 옆에 있는 **"확인 없이 열기"** 클릭
5. 다시 앱 더블클릭 → 마지막으로 한 번 확인 다이얼로그 → "열기"

---

## 권한 허용

앱이 처음 켜지면 두 가지 권한을 요청해요:

- **📷 카메라** — 자리 비움 감지용 (얼굴 인식만 함, 영상 저장/전송 없음)
- **🔔 알림** — 휴식 시간 시작/종료 알림용 (온보딩 4단계에서 요청)

둘 다 **허용**해주세요. 안 해도 앱은 돌아가지만 핵심 기능이 빠져요.

## 사용법

### 메뉴바 아이콘
설치 후 **메뉴바 우상단에 🍅 아이콘**이 생겨요. 클릭해서 모든 동작 가능.

### 첫 실행: 온보딩
앱이 처음 켜지면 설정 윈도우가 자동으로 열려요.

1. **Welcome** — 앱 소개
2. **안전 시야 범위 등록** — 평소 보시는 모니터의 양쪽 끝을 보면서 "현재 위치 등록" 클릭 (최소 2개)
3. **숨길 앱 등록** — 누가 다가오면 자동으로 숨겨질 앱 선택 (카톡, 슬랙 등) - 건너뛰기 가능
4. **알림 권한** — "알림 허용 요청" 클릭
5. **완료** — 시작!

### 평소 사용
- 메뉴바 🍅 클릭 → **시작** 버튼 → 50분 집중 타이머 시작
- 50분 끝나면 자동으로 10분 휴식, 다시 50분 작업 사이클
- 자리 비우면 타이머 자동 일시정지 (5초+ 얼굴 안 보이면)
- 자리 비우면 스크린 세이버 자동 실행 (얼굴 안 보일 때만)
- 뒤로 휙 돌아보면 등록한 앱 자동 숨김 + "쉬는 시간 종료" 알림

## 사용 팁

- **카메라가 너무 자주 자리 비움으로 인식**: 카메라 각도 점검 (얼굴이 화면에 잘 잡혀야 함)
- **트리거가 자주 잘못 발동**: 메뉴바 → 설정 열기 → safe range에 등록 위치 추가
- **알림이 안 옴**: 시스템 설정 → 알림 → Pomo → 알림 허용 확인
- **온보딩 다시 보고 싶음**: 메뉴바 → 설정 → 맨 아래 "고급" → "온보딩 다시 보기"

## 알려진 한계

- macOS 15 (Sequoia) 이상에서 보안 우회 절차 조금 더 복잡
- Apple Developer 인증 안 받은 unsigned 앱이라 첫 실행 마찰 있음
- 카메라 사용 중 표시(메뉴바 녹색 점) 못 끔 — macOS 강제 정책
- 자동 업데이트 없음 — 새 버전 나오면 zip 다시 받아야 함
