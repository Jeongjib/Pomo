# 🍅 Pomo

뽀모도로 타이머 + 자리 비움 자동 감지 macOS 앱.

- 50분 집중 / 10분 휴식 사이클
- 카메라로 자리 비움을 자동 감지 → 타이머 자동 일시정지 + 스크린 세이버
- 누가 다가오면 등록한 앱 자동 숨김 (선택)

---

## 다운로드 (사용자용)

[Releases 페이지](../../releases)에서 최신 `Pomo.zip` 다운로드.

## 설치

### 1. 압축 풀기
받은 `Pomo.zip`을 더블클릭 → 같은 폴더에 `Pomo.app` 생성.

### 2. Applications 폴더로 이동 (권장)
`Pomo.app`을 **응용 프로그램(Applications) 폴더**로 드래그.

### 3. 첫 실행 — ⚠️ 그냥 더블클릭하면 안 열려요

이 앱은 Apple 공식 인증을 거치지 않아서 macOS가 처음에는 차단해요. 한 번만 다음 절차로 우회하면 그 다음부터는 그냥 더블클릭으로 열립니다.

**일반적인 경우 (macOS 14 이하)**
1. `Pomo.app` **우클릭** → **"열기"** 선택
2. 다이얼로그가 뜨면 **"열기"** 한 번 더 클릭

**macOS 15 (Sequoia) 이상**
위 방법으로 안 열릴 수 있어요:
1. 한 번 더블클릭 시도 (차단 메시지 떠도 OK)
2. **시스템 설정 → 개인정보 보호 및 보안** 열기
3. 스크롤 내려서 **"Pomo이(가) 차단되었습니다"** 메시지 찾기
4. **"확인 없이 열기"** 클릭
5. 다시 앱 더블클릭 → "열기"

## 권한 허용

앱이 처음 켜지면 두 가지 권한이 필요해요:

- **📷 카메라** — 자리 비움 감지 (얼굴 위치만 감지, 영상 저장/전송 없음)
- **🔔 알림** — 휴식 시간 시작/종료 알림 (온보딩 4단계에서 명시적으로 요청)

둘 다 **허용** 해주세요. 안 해도 앱은 돌아가지만 핵심 기능이 빠져요.

## 사용법

### 메뉴바 아이콘
설치 후 **메뉴바 우상단에 🍅 아이콘** 생성. 클릭해서 모든 동작 가능.

### 첫 실행: 5단계 온보딩
앱이 처음 켜지면 설정 윈도우가 자동으로 열려요.

1. **Welcome** — 앱 소개
2. **안전 시야 등록** — 평소 보시는 모니터의 양쪽 끝을 보면서 "현재 위치 등록" 클릭 (최소 2개)
3. **숨길 앱 등록** — 누가 다가오면 자동 숨겨질 앱 선택 (카톡, 슬랙 등) - 건너뛰기 가능
4. **알림 권한** — "알림 허용 요청" 클릭
5. **완료**

### 평소 사용
- 메뉴바 🍅 클릭 → **시작** 버튼 → 50분 집중 타이머 시작
- 50분 끝나면 자동으로 10분 휴식, 다시 50분 작업
- 자리 비우면 타이머 자동 일시정지 (5초+ 얼굴 안 보이면)
- 자리 비우면 스크린 세이버 자동 실행
- 뒤로 휙 돌아보면 등록한 앱 자동 숨김 + "쉬는 시간 종료" 알림

## 동작 시나리오

| 상황 | 작업 중 (🍅) | 휴식 중 (☕️) |
| --- | --- | --- |
| 누가 다가옴 → 휙 돌아봄 | hide ✅ + 알림 ✅ | hide ✅ + 알림 ❌ |
| 시선 5초+ 멀어짐 | 타이머 일시정지 | 타이머 일시정지 |
| 얼굴 5초+ 안 보임 | 일시정지 + 스크린 세이버 | 일시정지 + 스크린 세이버 |
| 자리 복귀 | 자동 재개 | 자동 재개 |

## 빌드 (개발자용)

```bash
git clone https://github.com/yourusername/Pomo.git
cd Pomo
open HeadPosePoC.xcodeproj
```

Xcode에서 ▶️ Run.

**요구사항:**
- macOS 14 (Sonoma) 이상
- Xcode 15 이상

**기술 스택:**
- Swift + SwiftUI (MenuBarExtra)
- AVFoundation (카메라)
- Vision framework (face/pose detection)
- UserNotifications
